# Personal Portfolio Template

A minimal personal portfolio template built with plain HTML and CSS — no frameworks, no npm, no build step required.

## How to use

1. Clone this repo: `git clone [this-repo-url] my-portfolio`
2. Open `index.html` in your browser (or right-click → Open with Live Server in VS Code)
3. Search for every `TODO` comment in `index.html` and `style.css` and replace with your info
4. Push to GitHub and deploy on Vercel — it just works

## What to customize

- **`index.html`** — your name, title, bio, 3 projects, GitHub/LinkedIn links, email
- **`style.css`** — change `--color-accent` at the top to any color you like

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com), click "Add New Project", import your repo
3. Click "Deploy" — no settings to change
4. Your portfolio is live at a `*.vercel.app` URL
